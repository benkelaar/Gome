Lightwight implementation Log4J logger that uses a Canvas for output. This package contains a very limitied implementation of a logger, that it compatiple to Apache's Log4J to a large extent. When using the logger the application does have an additional canvas element the logging output is written into. The application can use Logger.debug(...), Logger.info(...), Logger.warn(...), Logger.error(...) or Logger.fata(...) as in any Log4J environment.
